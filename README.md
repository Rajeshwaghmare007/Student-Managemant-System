# 🎓 Student Management System (Java)

A simple and interactive **Student Management System** built using **Core Java**. This project is designed for beginners to understand Java programming, Object-Oriented Programming (OOP), ArrayList, and CRUD (Create, Read, Update, Delete) operations.

---

## 📌 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student
- ✏️ Update Student
- ❌ Delete Student
- 🚪 Exit Application

---

## 🛠 Technologies Used

- Java 21
- Core Java
- Object-Oriented Programming (OOP)
- Scanner Class

## ▶️ How to Run

### Compile

```bash
javac Main.java
```

### Run

```bash
java Main
